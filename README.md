# Cyclistic Power BI Dashboard
Para este proyecto, el objetivo se centraba en poder hacer uso de las características del programa Power BI para mejorar las capacidades de visualización; se eligió un proyecto conocido y trabajado previamente, el Caso de Estudio 1 - Cyclistic. 
* **Descripción:** Reporte de visualización de datos del dataset limpio trabajado en R. Se muestra el contraste y métricas utilizadas para comparar las diferencias en el uso diario del servicio de renta de bicicletas compartidas en la ciudad de Chicago.
* **Proceso:** Se comenzó por exportar el *script* de R como un archivo .csv, para posteriormente introducirlo en Power BI. Se generó una nueva tabla en la vista de tabla que funcionaría como una especie de calendario, extrayendo en columnas diferentes el año, mes y día para cada una de las filas del dataset. Se estableció una relación en la vista de modelo entre esta nueva tabla y la original y se procedió a generar visualizaciones comparando viajes totales, semanales, estaciones y horas de entrada entre miembros y usuarios casuales.
* **Herramientas Usadas:** R (exportación de archivo .csv), Power BI.
